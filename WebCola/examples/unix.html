Directed graph flow layout example

This is a cola.js rendering of the classic graphviz dot example. We use graphlib-dot.js to load the dot file. The initialization for cola here is interesting: 
cola
    .linkDistance(150)
    .avoidOverlaps(true)
    .directedLinkConstraints({ axis: 'x', gap: cola.linkDistance() })
    .size([width, height])
    .nodes(nodes)
    .links(edges)
    .jaccardLinkLengths();
In particular, the call to directedLinkConstraints causes all edges not involved in a cycle to have a separation constraint generated between their source and sink, with a minimum spacing set to the ideal linkDistance. This achieves a left-to-right flow layout. Change 'x' to 'y' to get top-to-bottom flow layout.
